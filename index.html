Basics of Web and HTML - 1.1

Quick Summary

Computers communicate using protocol called http. When a request is initiated from a browser, http will communicate this request to servers in the web. Documents containing content are sent back to the browser in a form of HTML, which is language used to wright websites.

HTML

HTML documents are made up of text content, markup, Tags and elements. Text content, describes what you see. Where markup describes how it looks.

There are 2 types of elements, Block and Inline elements making up an HTML document. Tags are used to tell the browser which elements are used.

Summary

Block: div, p, form

Inline: a, span, br, img, strong

Anchors: are used for links
